Tonal Extractor turns mood boards and references into actionable palette systems.

## Core functions

- Samples dominant and supporting tonal ranges.
- Generates contrast-aware token sets.
- Outputs metadata aligned with style guides.

## Planned updates

- Palette drift detection across revisions.
- Accessibility-first alternate palette generation.
