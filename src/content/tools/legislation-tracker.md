This tool helps follow legislative changes without manually checking multiple portals.

## Current capabilities

- Aggregates updates into one timeline.
- Groups records by jurisdiction and status.
- Flags state changes daily.

## Planned additions

- Email digest preferences.
- Saved filters per policy area.
